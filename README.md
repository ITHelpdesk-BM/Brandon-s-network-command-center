# Brandon’s Network Command Center

A lightweight IT network monitoring dashboard built with Python and Dash.

## Features
- Real-time service monitoring
- HTTP latency tracking
- Online / Offline status detection
- Historical logging
- CSV export for reporting
- Auto-refresh dashboard every 30 seconds

## Monitored Services (default)
- Google
- GitHub
- Cloudflare

## Tech Stack
- Python
- Dash (Plotly)
- Requests
- Pandas

## How to Run

```bash
pip install -r requirements.txt
python app.py
